Bubble sort:-
Take first number and compare with next, if number greater than current, swap with current number
continue for all numbers

Selection sort:-
Take first number and compare with others, find least number and then swap with current number
continue for all numbers

Insertion sort:-
Take second number, compare it with preceding number, if number greater than current, shift current number to start and shift preceding number
continue for all numbers

Quick sort:-
Take last element as pivot
find number from left > pivot
find number from right < pivot
if left < right then swap left and right
if left > right then swap left and pivot
